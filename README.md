## Project Overview
This is a complete Flutter application developed as final assignment.  
It includes authentication, API handling, state management, and offline support.

---

## Features
- User Login & Registration
- Dashboard with dynamic data
- Course management system
- API integration
- Offline data storage support
- State management using Provider

---

## Technologies Used
- Flutter SDK
- Dart
- Provider (State Management)
- HTTP (API calls)
- SharedPreferences (Local storage)
- Git & GitHub

---

## Project Structure
lib/
 ├── controllers/
 ├── models/
 ├── providers/
 ├── repositories/
 ├── screens/
 ├── services/
 ├── utils/
 └── main.dart

---

## Offline & State Management
- Data is fetched from API and stored locally
- SharedPreferences used for offline caching
- Provider manages UI state updates efficiently
